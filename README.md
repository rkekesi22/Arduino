# Arduino
Projekt: Gyalogos lámpa nyomógombbal

A projektet, Wokwi online szimulátorral készítettem. (https://wokwi.com/)

Egy kisebb forgalmi szituációt próbáltam meg ábrázolni. 

Alapból zöld a lámpa  a járműveknek, piros a gyalogosoknak

Ha megnyomod a gombot:

- Járműveknek
Zöld → Sárga → Piros

- Gyalogosoknak
Piros -> Zöld 

Kis várakozás

Vissza Zöldre, gyalogosoknak pirosra.
